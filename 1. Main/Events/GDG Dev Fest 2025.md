### Jules - Autonomous Code Agent

![[IMG_20251206_105305.jpg|Разница между ИИ и агентами]]

Jules - полностью автономный кодинг агент. Его презентуют как твоего личного Junior разработчика. Работать с ним нужно также как и с джуном. 
![[IMG_20251206_105454.jpg]]

Разница Jules от других ИИ агентов

![[IMG_20251206_105712.jpg]]

Можно дать ему задачи и он сделал всё сам, нужно только его сделать обзор кода (code review). Он живёт в облаке.

![[IMG_20251206_105952.jpg]]

Он не просто один агент, у него есть субагенты, которые помогают ему работать. Он сначала планирует то, что нужно сделать, разбивая цель на микроцели. Jules знаком с UI и принимает решения касаемо дизайна хорошо. Он сам делает Pull Request-ы.
Jules может работать прямо на Github и вносить изменения прямо в мой репозиторий.

#### Future goals for Jules 

![[IMG_20251206_110705.jpg]]

У него есть несколько субагентов, которые влияют на качество кода.

Думай о code агентах как об источнике знаний, а не просто как о "black box". Спикер использует code агентов на своей работе и говорит, что они помогли ему стать продуктивнее.

![[IMG_20251206_111047.jpg]]

Они хотят продвинуть Jules до уровня Senior разработчика.
