### Экосистема Asterium 

![[IMG_20251113_193006.jpg]]

У них есть свой токен - Humo Token на своём блокчейне. 1 Humo Token = 1000 сум. Также у них есть криптокарта Humo, не привязанная ни к какому банку.

**Эквайринг** - возможность оплачивать покупки с помощью любого криптокошелька, будь то Metamask, Trust Wallet и т.д. Запускается со следующего года.

Планируют коллаборацию с разными банкоматами. Либо с помощью ПО, либо с помощью своего отдельного банкомата.

![[IMG_20251113_193456.jpg]]

**Криптодепозитарий** - ???

Выпустили виртуальную карту Visa Platinum. Какая-то особенная карта, которая может хранить крипту. Также есть какая-то humo daily. Открыть карту humo daily стоит 5\$, Visa Platinum 10$.

### Обмен крипты на суммы 

Обмен проходит по предварительной записи. Сначала нужно записаться в офисе, а потом уже обменивать. Причина - большие очереди. Они постепенно открывают филиалы по всему городу.

Закон пока не позволяет получать проценты с хранения крипты. Возможно это реализуют позже, но пока сказать точно нельзя. 

Для оплаты криптокартой нужно сначала перевести крипту с кошелька в приложении Asterium, потом крипта конвертируется в суммы и переносится на криптокарту.

Криптобиржы хранят свои средства на нескольких холодных или горячих кошельках.
