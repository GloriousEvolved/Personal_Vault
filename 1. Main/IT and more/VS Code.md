### Extensions:
1) `Auto Rename Tag` - при изменение открывающего тега в HTML сразу меняет и закрывающий тег.
<br/>
2) `Auto Close Tag` Jun Jan - автоматически вставляет закрывающий тег.
<br/>
3) `Better Comments` Aaron Bond - улучшенные комментарии.
<br/>
4) `eCSStractor` - копирует теги из HTML, чтобы вставить в CSS по команде `CTRL + ALT + E`.
<br/>
5) `Gemini Code Assist` - ИИ помощник.
<br/>
6) `GitHub Repositories` - контроль GitHub репозиториев.
<br/>
7) `Remote Repositories` - тоже что-то связанное с репозиториями.
<br/>
8) `htmltagwrap` - с помощью горячих клавиш `ALT+W` оборачивает выделенную область в тег.
<br/>
9) `Image preview` - предпросмотр изображений в HTML.
<br/>
10) `indent-rainbow` - подсветка отступов.
<br/>
11) `Live Sass Compiler` - компилятор для scss кода. С помощью него scss файлы преобразуются в css файлы.
<br/>
12) `Live Server` - запускает HTML файлы на локальном сервере и не нужно постоянно обновлять страницу, чтобы увидеть изменения. Live Server не будет автоматически обновлять сайт если в названии папки с файлами присутствует `.` или `#`. Работает с круглыми скобками `()`, в них уже можно поставить `#`. Будет выглядеть следующим образом `(#) Имя папки`.
<br/>
13) `Material Icon Theme` - добавляет красивые иконки к папкам, файлам и картинкам.
<br/>
14) `Prettier` - форматет кода.
<br/>
15) `Svg Preview In Code` Izp - предпросмотр svg иконок в VS Code, похож на `Image preview`.
<br/>
16) `Tokyo Night` enkia - моя тема для VS Code.
### Hotkeys

^ede582

`ALT+⬆ ALT+⬇` - меняет местами строки кода вверх или вниз
`SHIFT+ALT+⬆ SHIFT+ALT+⬇` - копирует код вверх или вниз
`CTRL+H` - заменяет выделенный код во всех местах в файле ^4bbafc
`CTRL+/` - однострочный комментарий.
`CTRL+SHIFT+/` - многострочный комментарий.
`log` - при написании `log` и нажатии пробел автоматически пишется `console.log()` ^cbd286
