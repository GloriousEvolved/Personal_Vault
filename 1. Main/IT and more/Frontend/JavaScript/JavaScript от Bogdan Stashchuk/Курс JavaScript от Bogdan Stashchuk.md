Переменные дают возможность повторного доступа к значениям. Переменные можно воспринимать как контейнеры.

#### Имена переменных [43:57](https://www.youtube.com/watch?v=CxgOKJh4zWE&t=2637s)

^9eb810

Имена переменных записываются несколькими способами:
1. *PascalCase* - в этом способе название начинается с заглавной буквы `Pascal`, другие слова, являющиеся частью переменной, тоже начинаются с заглавной буквы - `Case`. Все слова пишутся слитно PascalCase используется для того, чтобы называть типы и классы(array - массив).
2. *DB_PASSWORD* -  в этом способе всё с заглавной буквы и между словами используется нижнее подчёркивание.  DB_PASSWORD  используется для переменных, значение которых известно до запуска приложения и эти значения не меняются(например для пароля к базе данных database password)
3. *camelCase* - в этом способе название первая буква маленькая, а остальные слова начинаются с заглавной буквы и все слова пишутся слитно. Этот способ используется во всех остальных случаях и чаще всего.

<span style='text-decoration: underline; font-weight: bold; color:7dcff'>Название переменных должны быть понятным, в первую очередь самому себе. Нужно называть переменные так, чтобы они отражали значения переменных или то, что они делают.</span>

#### Объявление переменных [49:43](https://www.youtube.com/watch?v=CxgOKJh4zWE&t=2983s)

^0cd305

Есть 3 слова, для объявления переменных: `let`, `const` и `var`.

`let` и `const` появились в ECMAScript 6(он появился в 2015 году). `var` - тип устаревший и его лучше не использовать.

```javascript
let a - это объявление переменной

const c = 10 - это объявление и присвоение переменной

a = true - это только присваивание значения, переменная a была объявлена в начале, но значение присвоили ей в конце
```

#### Примитивные типы [1:08:59](https://www.youtube.com/watch?v=CxgOKJh4zWE&t=4139s)

^ea6a63

В <span style='color: EFD81D;'>JavaScript</span> есть 6 примитивных типов.
Основные типы:
1. <span style='color: 0BADF4'>string (строка) - это текст</span>
2. <span style='color: 0FBCA6'>boolean (логический) - это правда(true) или ложь(false),</span> 
3. <span style='color: 46D57E'>number (число) - любые числа</span>
4. <span style='color: 0A7EAB'>undefined - есть только одно значение undefined. Значение подразумевает то, что у переменной нет значения, оно не существует, оно не определенно</span> ^1d1901
   
   Эти типы используются редко:
5. <span style='color: DF5F7D'>null - есть только одно значение null. Используется, когда я хочу явно указать, что у какой-то переменной значение отсутсвует</span> ^d6ce1a
6. <span style='color: 08874A'>symbol (символ) - это тип, для создания уникальных значений(вроде бы не важный тип, так как используется редко)</span>
