```javascript
console.log('Hello World'); - это выражение.
```

`console` - это [[Самое важное в JavaScript#^1d0019|объект]].
`log` - это [[#^0fd74a|метод]].
`.` - это [[#^8356de|точечная запись]].
`()` - это вызов [[#^0fd74a|метода]]. Мы вызываем функцию, которая является методом объекта `console`.
`Hello Wolrd` - это аргумент, значение типа [[string(строка)]].

> [!question] Что такое метод?
>  **Метод** - это функция, которая является значением одного из свойств объекта. Если значением ключа будет функция, то такое [[#^6a4aa2|свойство]] будет называться **методом**. ^0fd74a

То есть,  у нашего объекта `console` есть свойство, значение которого - это функция, а такие свойства называются [[#^0fd74a|методами]].

> [!question] Точечная запись
>  **Точка `.`** - это синтаксис JS, который называется *точечная запись*. С помощью точечной записи можно получать доступ к [[#^6a4aa2|свойствам]] объекта.  ^8356de

В данной случаем мы получаем, с помощью [[#^8356de|точечной записи]], доступ к методу объекта - метод называется `log`.

Аргумент в данном примере - это просто значение типа [[string(строка)]].

Методы `console`:

```javascript
console.log(); - описан выше ↑

console.dir(); - отображает все свойства объекта.

console.table(); - отображает все свойства объекта в табличном виде.
```
