 `=` - это оператор присваивания. ^1d7370

```javascript
a = 5

b++

myFunction(c,d)
```

Оператор присвоения в данном примере используется в выражении, то есть значение <span style='color:bb9af7; font-family:Roboto Mono;'>20</span> присваивается переменно <span style='color:7dcff'>а</span>

#### Выражения с побочными действиями

> [!question] Что такое выражение с побочными действиями?
>  **Выражения с побочными действиями** - выражение, которое не только возвращает значение, но и выполняет какие-то действия(side effects/побочные эффекты):

```javascript
a = 5

b++

myFunction(c,d)
```

`a` <span style='color:ff757f'>=</span> <span style='color:bb9af7; font-family:Roboto Mono'>5</span> - в этом выражении используется [[#^1d7370\|оператор присваивания]]. Это выражение вернёт значение - 5, но кроме этого, оно приведёт к выполнению действия, а именно - это значение 5 будет присвоено к переменной a, и позже к ней можно будет обратиться и прочесть её значение. Мы выполнили побочные действия(side effect).

`b`<span style='color:ff757f'>++</span> - здесь используется [[++]]. Значение после этого оператора значение увеличится на единицу. Выражение вернёт цифру, допустим 7, и кроме этого значение переменно `b` увеличится на единицу. То есть помимо возврата нового значения, оно увеличится и присвоится переменной `b`.

<span style='color:ffc777'>myFunction</span>(c,d) - в этом примере, в функции можно выполнять очень много побочных действий, например, обращение к другому серверу и получение от него данных, чтение данных с файла и так далее, помимо возврата значения.
