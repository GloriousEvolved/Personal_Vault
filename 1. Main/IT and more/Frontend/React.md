React CDN - для частичного использования React в проекте.
React CLI - для создания проекта полностью на React.
## React cli

1. Открыть консоль через `WIN + R` и в ней прописать `cmd`.
2. В консоли прописать команду `node -v` для проверки версии node.js.
3. При необходимости скачать и обновить *node.js* с официального сайта.
4. Далее зайти в папку где хотим создать проект. В пути написать `cmd` и нажать `Enter`.
5. Прописать `npm create vite@latest` - команда для создания проектов. Если вышло подтверждение, пропиши `Y` и нажми Enter. Дальше прописать название проекта(папки), выбрать *React* и дальше выбрать *JavaScript SWC*. `vite` - это сборщик проектов, а `@latest` означает послденяя версия. *JavaScript SWC* - какой-то ускоренный *JavaScript*.
6. После этого в выбранной папке появится папка с базовой структурой *React* приложения.
7. В терминале внутри *VS Code* нужно прописать `npm i` - чтобы установить *node* модули. Без *node* модулей проект не запустится.

`npm run dev` - команда для запуска проекта. Выдаёт ссылку на которую можно кликнуть и открыть свой проект.

`CTRL + C` - команда для остановки проекта и команды `npm run dev` чтобы я мог писать команды в терминале. Её нужно вписать внутри терминала. 

Для корректной работы `scss` нужно установить модули с помощью следующих команд в терминале: `npm i sass` и `npm i sass-loader`.

Для скачивания библиотеки `React Router`, нужно в терминале прописать: `npm i react-router-dom`. Далее в файле `main.jsx` нужно внутри `<StrictMode></StrictMode>` прописать `<BrowserRouter></BrowserRouter>`, а внутри него уже `<App />`.

Для работы с cookies нужно установить пакет с помощью команды `npm i js-cookie`. Если работаешь с TypeScript, то нужно установить `npm i @types/js-cookie`.
