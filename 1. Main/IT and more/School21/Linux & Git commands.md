### Linux
`CTRL+H` - отображение скрытых файлов в файловом менеджере.

### Terminal:

`ls` - показать список файлов.
`cd название директории` - перемещение между директориями.
`cp [имя файла для копирования] [путь куда нужно скопировать]` - создание копии файла. Если нужно сделать копию, то можно просто написать старое название файла и новое.
`mkdir название файла` - создание папки.
`touch файл` -  создание файла.
`cat файл` - просмотр содержимого файла.
`ps` - отчёт о работающим процессах.
`ps -a` - вывод всех процессов.
`pkill имя процесса из ps` - останавливает запущенный процесс по его имени.
`man команда` (manual) - подробное описание команды.
`mv [файл] [путь]` - перемещение файлов. С помощью него, вроде тоже можно создать копию.

`>` - перезапись файла с удалением старого контента внутри.
`>>` - добавление информации в конце файла.
`readlink -f` узнать абсолютный путь файла.
`stat` отображение подробных метаданных файла.

`CTRL+L` - очистка терминала.

`git checkout имя ветки` - переход на ветку.
`git checkout -b имя ветки` - создание новой ветки.
