`scanf("")` - функция из библиотеки <stdioh>, которая считывает данные из потока ввода и форматирует их. Проще говоря получение данных с клавиатуры/input. Для указания форматирования нужно в круглых скобках указать спецификатор `%d` - формат целочисленных чисел. 

`&` - указатель для переменных.

Объявляй переменные там, где их используешь. Объявление всех переменных в начале скрипта/кода называется *Pascal Style*, так как иначе в Pascal нельзя.

Функция с типом `void` не возвращает никакое значение. Она является процедурой, может что-то делать, но не возвращает значение.

### Условия

```c
while (условие) { 
  код/инструкция // Проверяет условие, а затем выполняет код
}
```
```c 
do {
  код/инструкция // Сначала выполняет код, а потом проверяет условие
}
while (условие) {
```

```c
for (переменна; условие; итерация) { 
  код/инструкция // выполняет код, пока условие истинно/соблюдается.
}
```

### Динамическая память 

Для работы с динамической памятью нужно подключить библиотеку stdlib.h

Выделяют 4 основные функции работы с динамической памятью: [[calloc()]], [[malloc()]], [[realloc()]],[[free()]];
```c
#include <stdlib.h>

(тип данных*)malloc() // memory allocation - выделение памяти под массивыю.
calloc() // contigious allocation - заполнение памяти нулямию.
realloc() // re-allocation - изменение выделенной памяти, то есть увеличивание или уменьшениею.
free(созданный указатель) // Освобождение памяти.
```
