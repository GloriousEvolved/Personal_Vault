*Ссылка:* [Что выяснили про ChatGPT: первые реальные данные несколько удивляют / Хабр](https://habr.com/ru/articles/947598/)

<br>

1. ChatGPT стал народным инструментом, а не игрушкой для программистов.

2. Люди используют его в основном для жизни, а не для работы.

3. Женщины обогнали мужчин в использовании. На начало 2023 года 80% пользователей - мужчины(подробнее смотри таблицу ниже).

4. Программирование — это миф, всего 4% запросов.

5. Образование важно — более образованные люди чаще используют для работы.

6. Бедные страны растут быстрее богатых — AI действительно выравнивает возможности.


| Показатель                                      | Значение              |
| ----------------------------------------------- | --------------------- |
| Пользователей в неделю                          | 700+ млн              |
| Личных запросов                                 | 73%                   |
| Рабочих запросов                                | 27%                   |
| Женщин пользователей                            | 52%                   |
| Мужчин пользователей                            | 48%                   |

| Процент запросов, связанных с программированием | 4.2% от всех запросов |
